# Платформа Взаимопомощи — Официальное описание проекта

## 1. Обзор проекта
Платформа Взаимопомощи — это структурированная система, созданная для того, чтобы люди могли помогать друг другу в достижении жилищных целей. Участники делают добровольные взносы (10€, 50€, 100€), а собранные средства направляются одному участнику согласно прозрачной очереди.

Принцип работы основан на непрерывной взаимной поддержке: после получения помощи участник остаётся в проекте и поддерживает следующих.

## 2. Цель проекта
Проект создан для людей, которые:
- не могут накопить на жильё самостоятельно,
- не имеют финансовой поддержки семьи,
- ищут прозрачный и понятный механизм взаимопомощи,
- хотят видеть открытое движение очереди,
- хотят участвовать в стабильной системе.

Проект поддерживает только цели, связанные с жильём.

## 3. Модель участия
1. Участник подаёт заявку, указывая свою цель.
2. Делает добровольный взнос (10€, 50€, 100€).
3. Добавляется в открытую очередь.
4. Один из участников получает помощь согласно своей позиции.
5. После получения помощи участник остаётся в проекте и продолжает помогать другим.

Модель работает за счёт постоянного добровольного участия сообщества.

## 4. Основные принципы
- Добровольность участия.
- Прозрачная очередь и открытая отчётность.
- Некоммерческий характер: нет кредитов, процентов, инвестиций.
- Цели строго жилищные.
- Минимальная комиссия — только на техническую поддержку и обслуживание.
- Полная версия сайта на русском и английском языках.

## 5. Отчётность
Платформа публикует:
- поступающие взносы,
- пожертвования,
- оказанную помощь,
- использование комиссии,
- движение очереди.

Вся информация доступна публично.

## 7. Технические характеристики
- Статический HTML/CSS сайт.
- Размещение: Netlify.
- Google Analytics 4 (ID: G-S002SKDYR4).

- поддержки работы сайта,

Цели должны относиться только к сфере жилья.
